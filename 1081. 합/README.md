# 1081. 합

### 문제
L보다 크거나 같고, U보다 작거나 같은 모든 정수의 각 자리의 합을 구하는 프로그램을 작성하시오.

### 입력
- 첫째 줄에 두 정수 L과 U이 주어진다.

### 출력
- 첫째 줄에 문제의 정답을 출력한다.

### 제한
```
0 ≤ L ≤ U ≤ 2,000,000,000
```
### 예제 입력 1 
```
0 3
```
### 예제 출력 1 
```
6
```
### 예제 입력 2 
```
10 14
```
### 예제 출력 2
```
15
```
### 예제 입력 3 
```
0 0
```
### 예제 출력 3 
```
0
```
### 예제 입력 4
```
14 53
```
### 예제 출력 4 
```
296
```
### 예제 입력 5 
```
24660 308357171
```
### 예제 출력 5 
```
11379854844
```
